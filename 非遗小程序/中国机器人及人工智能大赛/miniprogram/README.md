# 中华文脉·AI共创工坊 - 微信小程序

## 项目简介

中华文脉·AI共创工坊是一款基于AI技术的传统文化共创平台，旨在通过人工智能技术降低传统文化创作门槛，激发大众参与热情，推动中华优秀传统文化的传承与创新。

## 功能模块

### 1. 首页
- 轮播图展示
- 核心功能入口（AI文生图、AI图生图、文化资源库、创作者社区）
- 热门创作推荐
- 文化资源推荐
- 月度创作挑战

### 2. 文化资源库
- 传统文化分类浏览
- 资源搜索功能
- 资源详情展示

### 3. AI创作工具
- 文生图功能（通过文字描述生成图片）
- 图生图功能（基于上传图片进行风格转换）
- 多种传统艺术风格选择
- 创意度和分辨率调整
- 生成结果保存和分享

### 4. 创作者社区
- 作品展示和浏览
- 作品点赞、评论和分享
- 关注作者功能
- 创作挑战参与

### 5. 个人中心
- 用户信息管理
- 我的作品管理
- 收藏管理
- 关注管理
- 系统设置

## 技术实现

### 前端技术栈
- 微信小程序原生框架
- WXML + WXSS + JavaScript

### AI技术
- 多模态AI生成技术（文生图、图生图）
- 传统艺术风格迁移算法

### 数据存储
- 微信云开发（可选）

## 运行方式

### 1. 开发环境配置
- 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信公众平台账号，获取小程序AppID

### 2. 导入项目
- 打开微信开发者工具
- 点击"导入项目"
- 选择项目目录：`c:\Users\马健波\OneDrive\Desktop\中国机器人及人工智能大赛\miniprogram`
- 输入AppID（或使用测试号）
- 点击"导入"按钮

### 3. 预览和调试
- 点击"预览"按钮，在模拟器中查看效果
- 点击"真机调试"按钮，在手机上进行测试

## 功能使用说明

### AI创作工具
1. 选择创作模式：文生图或图生图
2. 输入文字描述（文生图）或上传图片（图生图）
3. 选择艺术风格
4. 调整创意度和分辨率
5. 点击"生成图片"按钮
6. 生成完成后可以保存或发布作品

### 文化资源库
1. 通过分类浏览传统文化资源
2. 使用搜索功能查找特定资源
3. 点击资源卡片查看详情

### 创作者社区
1. 浏览热门、最新或关注作者的作品
2. 对作品进行点赞、评论和分享
3. 关注喜欢的作者
4. 参与月度创作挑战

## 注意事项

1. 本项目使用模拟数据，实际部署时需要接入真实的AI API和云数据库
2. 部分功能需要用户授权（如保存图片、获取用户信息等）
3. 建议在微信开发者工具中开启"不校验合法域名"选项进行本地开发
