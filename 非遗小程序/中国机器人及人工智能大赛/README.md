# 中华文脉·AI共创工坊 - 微信小程序项目

## 项目简介

"中华文脉·AI共创工坊"是一款基于微信小程序的AI驱动文化共创平台，旨在通过人工智能技术降低传统文化创作门槛，激发大众参与热情，构建一个开放、共享、可持续的文化创新生态系统。

本项目是为第二十八届中国机器人及人工智能大赛智能文化创意创新赛道设计的参赛作品。

## 核心功能

### 1. AI创作工具
- **AI文生图**：通过文字描述生成传统风格的图像作品
- **AI图生图**：将现代图像转换为传统艺术风格（国画、剪纸、水彩等）
- **多种艺术风格**：支持国画、剪纸、水彩、油画、素描等多种传统艺术风格
- **参数调整**：可调整创意度、分辨率等生成参数

### 2. 文化资源库
- **非遗数字馆**：收录全国各地区非遗项目资料
- **文化元素库**：整理传统纹样、色彩、符号等可复用元素
- **分类浏览**：按民间工艺、传统绘画、传统音乐、古典文学、传统建筑等分类
- **搜索功能**：支持关键词搜索传统文化资源

### 3. 创作者社区
- **作品展示**：展示用户创作的文化作品
- **互动功能**：支持点赞、评论、分享等社交互动
- **创作挑战**：定期举办主题创作挑战赛
- **关注系统**：支持关注其他创作者

### 4. 个人中心
- **用户信息管理**：微信授权登录，管理个人信息
- **作品管理**：查看和管理个人创作的作品
- **收藏管理**：管理收藏的作品和文化资源
- **关注管理**：查看和管理关注的创作者

## 技术架构

### 前端技术
- **微信小程序原生框架**：使用WXML、WXSS、JavaScript开发
- **UI设计**：融合传统美学与现代设计风格
- **响应式布局**：适配不同屏幕尺寸的设备

### 后端技术
- **微信云开发**：云函数、云数据库、云存储
- **数据库**：MongoDB（非结构化数据）+ MySQL（结构化数据）

### AI技术
- **大语言模型**：基于开源大模型进行传统文化领域微调
- **多模态模型**：文生图（Stable Diffusion）、图生视频
- **计算机视觉**：图像分割、风格迁移等技术

## 开发与使用

### 开发环境
1. 安装微信开发者工具
2. 注册微信公众平台账号并获取AppID
3. 克隆项目到本地
4. 在微信开发者工具中导入项目
5. 配置云开发环境（可选）

### 使用方法
1. 打开微信开发者工具，导入项目
2. 点击"编译"按钮，在模拟器中查看效果
3. 连接手机进行真机调试
4. 提交审核并发布到微信小程序平台
